# GUI / UX dokumentáció – Healthy Eating App

## Áttekintés

Ez a dokumentáció a Healthy Eating App felhasználói felületét (GUI) és felhasználói élményét (UX) mutatja be.

Az alkalmazás célja az egészséges életmód támogatása:
- receptek böngészése
- étkezések nyomon követése
- étrend tervezése
- bevásárlólista generálása
- edukációs cikkek megjelenítése

A projekt React, Node.js és MySQL technológiákkal készült.
